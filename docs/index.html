DEPLOYMENT

1

Clone the repository

git clone https://github.com/wanderer-industries/wanderer-ops.git
cd wanderer-ops
2

Configure environment

# Update env setting in docker-compose.yml
3

Start with Docker Compose

# Build and start all services
docker compose build
docker compose up -d
# View logs
docker compose logs -f wanderer-ops
# Check status
docker compose ps
4

Access the application

# Application will be available at: http://localhost:8001
# Default admin credentials (change in production!):
Username: admin
Password: (set via WANDERER_OPS_ADMIN_PASSWORD)

Security Notice

  • > Always generate a new SECRET_KEY_BASE for production
  • > Set strong WANDERER_OPS_ADMIN_PASSWORD
  • > Use SSL/TLS in production environments
  • > Restrict database access to application container only
